Features:

1.User Management
Register with email and password
Login with validation
User data stored in users.txt

2.Cart Management
Each user gets a separate cart file
Cart file name format:
cart_user_at_example.com.txt
Automatically loads saved cart on login
Automatically saves cart on exit or adding items

3.Shopping System
Preloaded product list:
Jeans – ₹1200
Shirt – ₹800
Pants – ₹900
T-Shirt – ₹500
Shorts – ₹400
Ability to add multiple items with quantity
Cart updates if same item is added again

4.Checkout System
Shows full bill with:
Item
Quantity
Price
Subtotal
Total amount
Option to Pay or Save for Later

5.File Handling Concepts Used
Reading/Writing .txt files
Creating user-specific files dynamically
Splitting/Joining CSV lines manually
Basic error/validation handling
